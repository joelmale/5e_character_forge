## Layout Options

The character sheet offers three distinct layout modes, automatically adapting for mobile devices:

### Modern Stacked Layout (Default)
- **Description**: Clean, organized sections with collapsible panels
- **Best For**: Desktop use, comprehensive character management
- **Features**: Drag-and-drop panel reordering, smart collapse defaults

### Classic D&D Layout
- **Description**: Traditional stat block format inspired by official D&D materials
- **Best For**: Familiar D&D styling, print-friendly appearance
- **Features**: Compact information display, official-style formatting

### Mobile Layout
- **Description**: Touch-optimized interface for phones and tablets
- **Best For**: Mobile gaming, quick access to essential stats
- **Features**: Large touch targets, simplified navigation

**Layout Switching**: Use the layout selector in the top-right corner to switch between modes. Your preference is automatically saved.

## Core Character Information

### Character Header
- **Name & Level**: Character name, class, subclass, level, and background
- **Race & Alignment**: Racial information and moral alignment
- **Experience Points**: Current XP and progress toward next level
- **Portrait/Avatar**: Custom character portrait (if set)

### Character Statistics Panel
- **Armor Class (AC)**: Current AC with breakdown of sources
- **Hit Points**: Current/maximum HP with temporary HP tracking
- **Initiative**: Initiative modifier with advantage/disadvantage options
- **Proficiency Bonus**: Current proficiency bonus (+2 to +6)
- **Passive Perception**: Automatically calculated passive Perception score
- **Speed**: Base movement speed with any modifiers

## Ability Scores & Checks

### Ability Score Display
- **STR/DEX/CON/INT/WIS/CHA**: All six ability scores with modifiers
- **Score Breakdown**: Base score + racial bonuses + other modifiers
- **Modifier Display**: Automatic calculation of ability modifiers (-5 to +10)

### Ability Checks
- **Click to Roll**: Click any ability score to make an ability check
- **3D Dice Animation**: Realistic dice rolling with physics simulation
- **Roll Results**: Detailed breakdown of roll components
- **Advantage/Disadvantage**: Right-click for advantage/disadvantage rolls
- **Roll History**: Access recent rolls via the dice tray

## Skills & Saving Throws

## Quick Reference

### Keyboard Shortcuts
- **Ctrl+R**: Roll initiative
- **Ctrl+H**: Take short rest
- **Ctrl+L**: Take long rest
- **Space**: Open dice tray
- **Esc**: Close modals

### Common Workflows
1. **Combat Setup**: Set initiative, check AC, prepare attacks
2. **Spell Preparation**: Open spell preparation modal, select spells
3. **Level Up**: Click level up button, follow guided process
4. **Equipment Management**: Use equipment browser, equip items
5. **Rest Recovery**: Choose rest type, manage resources

### Troubleshooting
- **Missing Features**: Check if sections are collapsed
- **Incorrect Calculations**: Verify ability scores and proficiencies
- **Layout Issues**: Try different layout modes
- **Performance Problems**: Close unused browser tabs

For additional help, refer to the [Character Creation Guide](character-creation.md) or [Monster Library Guide](monster-library.md).
